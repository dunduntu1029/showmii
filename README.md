# showmii 靜態商店

這是一個純靜態商品展示網站。訪客可以切換繁體中文、简体中文、English；購買方式是「聯繫管理員」，不接付款、不產生訂單。

## 進入管理模式

網站上不會顯示後台入口。你自己用下面其中一種方式進入：

1. 在網址最後加上 `#manage`

```text
https://你的帳號.github.io/你的專案/#manage
```

2. 或在網站畫面按：

```text
Ctrl + Alt + M
```

預設密碼：

```text
showmii-admin
```

## 最簡單的編輯方式

進入後台後，第一個分頁就是 `快速設定`。

在這裡可以直接改：

- 店名
- 顯示店名
- Logo 字母
- 管理密碼
- 預設語言
- 公告文字
- 首頁小標、大標題、說明、按鈕文字
- Facebook / Instagram / LINE 等外部聯絡網站

## 編輯聯絡購買方式

在 `快速設定` 或 `聯絡方式` 分頁都可以改。

每一行格式：

```text
名稱 | 網址 | 備註
```

例子：

```text
Facebook | https://www.facebook.com/你的粉專 | 私訊粉專詢問
Instagram | https://www.instagram.com/你的帳號 | IG 私訊詢問
LINE | https://line.me/R/ti/p/@你的LINEID | 加 LINE 詢問
```

## 編輯商品

1. 進入管理模式
2. 點 `商品`
3. 選商品
4. 修改商品名稱、價格、分類、庫存、標籤、說明、圖片網址
5. 可新增、複製、刪除、上移、下移

## 讓修改公開到 GitHub

後台修改會先存在你自己的瀏覽器。要讓所有訪客看到：

1. 進入管理模式
2. 點 `備份/匯入`
3. 點 `匯出 GitHub 資料檔`
4. 下載的檔案改名成：`default-data.js`
5. 用它取代 GitHub 專案裡的：

```text
assets/default-data.js
```

6. 重新上傳/提交到 GitHub

## 限制

這是純靜態網站。隱藏管理模式只是讓一般訪客看不到入口，不是真正安全的多人後台。若未來要會員、付款、訂單、跨裝置同步，就需要後端或資料庫。
